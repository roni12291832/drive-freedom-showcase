Tema WordPress profissional para landing page do curso Método MASD - Supere o Medo de Dirigir.

== Descrição ==

O Método MASD é um tema WordPress moderno e responsivo criado especificamente para a landing page do curso de superação do medo de dirigir da psicóloga Márcia Mattos.

Características principais:
* Design totalmente responsivo
* Otimizado para conversão
* Integração com Kiwify para pagamentos
* Seções modulares
* Sistema de cores customizado (Teal e Gold)
* Suporte a vídeos de depoimentos
* Performance otimizada
* SEO friendly

== Instalação ==

1. Faça o upload da pasta do tema para `/wp-content/themes/`
2. Ative o tema através do menu 'Aparência' no WordPress
3. Configure o link de compra em Aparência > Personalizar > Configurações MASD
4. Personalize as cores e textos conforme necessário

== Personalização ==

1. **Link de Compra (Kiwify)**
   Vá em: Aparência > Personalizar > Configurações MASD > Link de Compra
   Cole o link da sua página de pagamento Kiwify

2. **Preço do Curso**
   Vá em: Aparência > Personalizar > Configurações MASD > Preço do Curso
   Defina o preço exibido na página

3. **Foto da Márcia**
   Faça upload da imagem no WordPress Media Library
   Use o nome do arquivo: marcia-mattos.jpg

4. **Vídeos de Depoimentos**
   Faça upload dos vídeos para a pasta: /wp-content/themes/masd/assets/videos/
   Nomeie os arquivos como: testimonial-1.mp4, testimonial-2.mp4, etc.

== Template Parts ==

O tema usa template parts para organização:
* template-parts/content-painpoints.php - Seção de dores
* template-parts/content-benefits.php - Seção de benefícios
* template-parts/content-about.php - Sobre a Márcia
* template-parts/content-method.php - Sobre o método
* template-parts/content-testimonials.php - Depoimentos
* template-parts/content-pricing.php - Preços e CTA

== Funções Úteis ==

* `masd_get_purchase_link()` - Retorna o link de compra configurado
* `masd_get_price()` - Retorna o preço configurado
